# 🎮 RPG Text Adventure (Python)

Simple text-based RPG collection developed in Python.
The project contains two playable games focused on decision-making, combat mechanics, and interactive storytelling.

---

## 📌 Overview

This repository includes:

- **Dxrpg.py** – A classic hero journey RPG with character classes and D20-based combat.
- **RPG.py** – A structured adventure titled Eldor: The Dragon's Curse with multiple stages and challenges.

Both games run directly in the terminal.

---

## 🧙 Dxrpg.py — The Hero's Journey

**Features:**
- 3 Classes: Warrior, Mage, Archer
- Attributes: Strength, Agility, Intelligence, Endurance
- D20-based combat system
- Player-driven decisions
- Multiple possible endings

---

## 🐉 RPG.py — Eldor: The Dragon's Curse

**Features:**
- Story divided into stages
- Combat and survival elements
- Weapon system
- Mini-game challenge
- Final boss encounter

---

## ▶ How to Run

Make sure Python 3 is installed.

Run one of the files:

```bash
python Dxrpg.py
```

or

```bash
python RPG.py
```
