# 🎮 RPG - Aventura Interativa em Texto

Bem-vindo ao projeto de **dois jogos RPG interativos** desenvolvidos em Python! Escolha seu personagem, defina seus atributos e embarque em uma jornada épica cheia de decisões, combates e mistério.

---

## 📋 Sobre o Projeto

Este repositório contém **dois RPGs completos em Python**:

### 1️⃣ **Dxrpg.py** - A Jornada do Herói
Um RPG focado em **narrativa imersiva** e **escolhas que importam**.

**Características:**
- 3 classes: **Guerreiro**, **Mago**, **Arqueiro**
- Sistema de atributos: Força, Agilidade, Inteligência, Resistência
- Decisões: Caçar vs Beber água (frutas como fallback)
- **Dois finais**: Morte ou Vitória épica
- Sistema D20 + atributos

### 2️⃣ **RPG.py** - Eldor: A Maldição do Dragão
História cinematic em 3 dias com múltiplas etapas.

**Características:**
- Mini-game de matemática
- Compra de armas
- Sobrevivência na selva
- Infiltração no castelo
- Missão para salvar a Princesa Elara

---

## 🚀 Como Jogar

### Execução

**Dxrpg.py:**
```bash
cd c:\Users\desenvolvedor\Desktop\rpg
python Dxrpg.py
```

**RPG.py:**
```bash
python RPG.py
```

---

## 🎯 Mecânicas

- **Atributos**: Força, Agilidade, Inteligência, Resistência
- **Sistema D20**: Rol de dados + média de atributos
- **Decisões**: Impactam diretamente no resultado
- **Finais Múltiplos**: Morte ou Vitória
